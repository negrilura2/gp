# 毕业设计说明书大纲
论文题目 ：基于声纹识别的智能门禁系统的设计与实现

## 第1章 绪论
### 1.1 选题背景与意义
- 1.1.1 传统门禁系统的局限性与生物特征识别的兴起。
- 1.1.2 声纹识别技术的优势（非接触、隐私性好、防伪造潜力）。
- 1.1.3 智能家居交互方式的演变（从机械开关到自然语言控制）。
- 1.1.4 本课题的研究意义（构建低成本、高安全、强交互的纯软件智能门禁原型）。
### 1.2 国内外研究现状
- 1.2.1 声纹识别技术发展（GMM-UBM [ o bj ec tO bj ec t ] → i-vector [ o bj ec tO bj ec t ] → x-vector [ o bj ec tO bj ec t ] → ECAPA-TDNN ）。
- 1.2.2 伪造语音攻击与防御技术现状（Replay Attack, Deepfake）。
- 1.2.3 智能门禁系统的应用现状（嵌入式 vs 云端架构）。
### 1.3 本文主要研究内容
- 构建基于 ECAPA-TDNN 的高精度声纹识别模型。
- 设计 端云协同 的智能控制架构（本地指令 + 云端 Agent）。
- 实现基于 WebSocket 的实时音频流处理与 VAD （语音活动检测）。
- 开发基于 Vue3 + Django + ChromaDB 的全栈门禁系统。

## 第2章 相关技术研究
### 2.1 声纹识别核心技术
- 2.1.1 语音信号预处理（预加重、分帧、加窗）。

- 2.3.1 WebSocket 全双工通信协议。

## 第3章 系统需求分析
### 3.1 业务流程分析
- 3.1.1 用户注册与声纹录入流程。
- 3.1.2 语音开门与身份验证流程。
- 3.1.3 管理员后台审计流程。
### 3.2 功能需求分析
- 用户端功能 ：
  - 实时波形显示与语音采集。
  - 声纹注册与模型更新。
  - 智能语音指令（开门、查询状态）。
- 管理端功能 ：
  - 用户权限管理（普通用户/管理员）。
  - 访问日志查询（时间、人员、置信度、意图）。
  - 系统状态监控。
### 3.3 非功能需求分析
- 实时性 ：端到端响应延迟 < 1.5秒。
- 准确性 ：声纹识别等错误率 (EER) 要求。
- 安全性 ：数据加密存储、防录音重放攻击（简述策略）。
## 第4章 系统设计
### 4.1 系统总体架构
- 采用 B/S 架构 ，划分为：前端感知层、网络传输层、后端服务层、数据持久层。
- 设计 “混合智能” (Hybrid Intelligence) 决策流：
  - 快路：本地规则匹配（高频指令）。
  - 慢路：云端 Agent 推理（复杂语义）。
### 4.2 核心模块设计
- 4.2.1 音频流处理模块 ：环形缓冲区设计、VAD 静音切除逻辑。
- 4.2.2 声纹认证模块 ：特征提取器接口、向量比对与阈值判决逻辑。
- 4.2.3 数据库设计 ：
  - E-R 图设计（User, UserProfile, VerifyLog）。
  - 向量集合设计（Collection: voice_prints ）。
### 4.3 接口设计
- WebSocket 信令定义（JSON 格式）。
- RESTful API 接口定义（用户管理、日志查询）。
